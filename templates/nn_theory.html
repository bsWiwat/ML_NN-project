Understanding the Fashion MNIST Dataset and CNN Architecture

Dataset Overview

Fashion MNIST Dataset

Data from : https://www.tensorflow.org/datasets/catalog/fashion_mnist

Fashion MNIST is a dataset of Zalando's article images, consisting of a training set of 60,000 examples and a test set of 10,000 examples.

Dataset Characteristics

  • 70,000 grayscale images
  • 28x28 pixels each
  • 10 fashion categories
  • Direct replacement for MNIST

Fashion Categories

  • T-shirt/Top
  • Trouser
  • Pullover
  • Dress
  • And more clothing items

Data Preparation Process

Image Preprocessing

  • Normalization (scaling pixel values to 0-1)
  • Reshaping images to match CNN input
  • Data augmentation techniques
  • Train-test split preparation

Data Augmentation

  • Random rotations
  • Width and height shifts
  • Zoom range adjustments
  • Horizontal flips

CNN Architecture

Layer Configuration

Convolutional Layers

  • Input shape: 28x28x1
  • First Conv2D: 32 filters
  • Second Conv2D: 64 filters
  • MaxPooling layers

Dense Layers

  • Flatten layer
  • Dense layer: 128 units
  • Dropout for regularization
  • Output: 10 units (Softmax)

Training Configuration

  • Optimizer: Adam
  • Loss: Categorical Crossentropy
  • Metrics: Accuracy
  • Early Stopping

Model Performance

Evaluation Metrics

  • Training accuracy and loss curves
  • Validation performance
  • Confusion matrix analysis
  • Per-class accuracy metrics
